## Модуль "Работа с БД (adminer)"
Модуль предназначен для работы с базой данных через библиотеку adminer (https://www.adminer.org/).

#### Возможности модуля
- Модуль Adminer;
- 31 шаблон дизайна панели на ваш выбор;
- Обновление до последней версии из интерфейса;
- Экспорт данных в форматах: SQL, CSV, TSV, XML, JSON, PHP Array, Markdown;
- Сжатие экспортируемых данных в форматах: gzip, zip, bzip2;
- Отображение php-сериализованных массивов в ячейках таблиц в человекопонятном виде;
- Всплывающие подсказки в фильтрации по столбцам;
- Фильтрация таблиц по имени;
- Автоматический доступ только для администратора без необходимости вводить реквизиты доступа;
- Возможность доступа к БД в случае пустого пароля в реквизитах доступа (ограничение adminer);
- Высокий протестированный уровень безопасности.

#### Минимальные требования:
- Главный модуль Битрикс: 14.0.0, т.к. модуль использует функции ядра D7.

#### Дополнительные рекомендации
При работе на ваших серверах используйте HTTPS-протокол для повышенной безопасности.

#### Установка модуля
Скопировать код модуля в папку /bitrix/modules/bxleader.adminer
